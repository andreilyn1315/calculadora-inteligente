# Calculadora Inteligente 🧠

Aplicación web que resuelve ecuaciones matemáticas paso a paso, mostrando el proceso de forma clara para facilitar el aprendizaje.

## 🚀 Funcionalidades
- Ingreso de ecuaciones (ej: 2x + 3 = 7)
- Visualización de los pasos de resolución
- Interfaz simple e interactiva

## 🛠️ Tecnologías usadas
- HTML
- CSS
- JavaScript
- Python (en desarrollo)

## 📖 Cómo usar
1. Escribe una ecuación en el campo de entrada  
2. Haz clic en el botón "calcular"  
3. Observa los pasos de la solución en pantalla  

## 📌 Estado del proyecto
En desarrollo 🚧  
Actualmente los pasos se muestran de forma simulada. Próximamente se integrará la lógica real con Python.
