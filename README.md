# 🧠 Calculadora Inteligente

Una aplicación web que no solo resuelve ecuaciones…  
sino que **te enseña paso a paso como un profesor**.

---

## 🌐 Demo en vivo

👉 https://andreilyn1315.github.io/calculadora-inteligente/

---

## 🚀 ¿Qué hace esta app?

Esta calculadora permite introducir ecuaciones algebraicas (por ejemplo: `2x + 3 = 7`)  
y devuelve una solución explicada paso a paso, simulando el razonamiento humano.

---

## 🧩 Características

- 🧠 Resolución de ecuaciones lineales
- 📖 Explicación paso a paso tipo tutor
- ⚡ Respuesta rápida usando API
- 🎯 Interfaz simple y directa
- 💻 Funciona en cualquier dispositivo (web responsive)

## 🔄 ¿Cómo funciona?

1. El usuario escribe una ecuación
2. El frontend envía la ecuación al backend
3. El backend procesa la ecuación en Python
4. Se generan los pasos matemáticos
5. El frontend muestra los pasos uno por uno

---

## 🧠 Ejemplo

Entrada:

2x + 3 = 7


Salida:

Ecuación original: 2x+3=7
Restamos 3 en ambos lados
2x = 4
Dividimos entre 2
x = 2
